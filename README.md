# Endroll — Downloads

Official installer releases for [Endroll](https://endroll.ai) — the local-first AI video editor.
Your footage never leaves your machine.

**Latest download:** https://dl.endroll.ai (redirects to the newest release below)

## Verify your download

Each release ships a `SHA256SUMS.txt`. On Windows PowerShell:

```powershell
Get-FileHash .\EndRoll-Setup-<version>.exe -Algorithm SHA256
```

Compare the hash against the one in `SHA256SUMS.txt`.

> **Note on Windows SmartScreen:** current builds are not yet code-signed, so Windows
> may show a "Windows protected your PC" prompt. Click **More info → Run anyway**.
> Code signing is on the roadmap before general availability.
